# 言包路线图

## 0.3.0：复用本机言序（已完成）

- 六目标 standalone 独立发布，制品不再捆绑言序运行时；
- 安装器在下载前检查并复用用户已安装的言序 1.1.6 或更高版本；
- 发布流程在封装前和上传前双重禁止制品包含`yanxu`。

## 0.2.1：言序 1.1.6 稳定发行（已完成）

- 六目标 standalone 与专用言序 1.1.6 工程运行时共同发布。
- 完整 CI、不可移动版本标签、Release 草稿门禁、安装后复验和 SHA-256 形成闭环。
- 清单/锁文件原子写、并发项目锁、离线/vendor 恢复和安全制品校验由言序 1.1.6 共享核心提供。

## 0.2.0：纯言序工程工具（已完成）

- [x] 删除 Cargo/Rust 业务实现，命令层完全迁移到言序；
- [x] Unix、PowerShell、CMD 无业务启动器；
- [x] 版本化工程协议握手；
- [x] 格式 2 项目初始化、依赖增删与失败回滚；
- [x] install/update/check/test/run/build/clean；
- [x] 完整传递依赖 `tree/why` 和 `doctor`；
- [x] YXB 与当前平台独立应用构建；
- [x] 真正的 `outdated` 和 `update --dry-run` 更新计划；
- [x] 确定性 `pack`、离线 `vendor`、依赖 `audit` 和结构化 `doc`；
- [x] 工作区批量命令与增量构建缓存报告；
- [ ] 稳定 JSON Schema 和 LSP 工程调用验证。
